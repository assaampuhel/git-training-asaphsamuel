# Memory Box

## Scripts

- `npm run dev`: Starts the Tailwind CLI build process in watch mode.
- `npm run build`: Builds the CSS for production.

## File Structure

- `src/input.css`: The main CSS file where Tailwind directives are injected.
- `dist/index.html`: The main HTML file.
- `dist/style.css`: The compiled CSS file generated by Tailwind.
- `dist/script.js`: Your custom JavaScript file.
- `dist/assets/`: Directory for images, fonts, and other static assets.
